# saucedemo
Тестовое задание для позиции AQA Java

### Общая информация
Тестируемый продукт - https://www.saucedemo.com

Цель: Покрыть тест-кейсами основные сценарии авторизации

#### Должно быть минимум 5 тестов:
- Успешная авторизация
- Авторизация с задержкой загрузки страницы
- Авторизация с пустыми полями
- Авторизация заблокированного пользователя
- Неправильный пароль

### Что есть в проекте:
1. Тесты авторизации находятся в src/test/java/tests
2. Веб-страницы описаны в src/test/java/pageobject
3. Утилитный класс и тестовые данные описаны в src/test/java/utils
4. Абстракции для тестов и страниц находятся в src/test/java/base
5. Сложные компоненты веб-страницы находятся в src/test/java/data
6. Общая конфигурация для тестов находятся в src/test/java/config


### Как запустить тесты:
mvn clean test

### Как создать отчёт вручную:

mvn clean test

allure serve target/allure-results

### Как посмотреть последний готовый отчёт:
- перейти в директорию docs проекта (в корне проекта)
- найти файл index.html
- открыть этот файл
